C++ AMP create_marker sample

-Overview:
This is a simple example of using markers to output the progress of a multi-step algorithm to the user.
Please refer to the accompaning blog post for more information.

-Hardware requirements:
This sample requires DirectX 11 capable card, if none detected sample will use DirectX 11 Reference Emulator.

-Software requirements:
Install Visual Studio 2012 from http://msdn.microsoft.com
