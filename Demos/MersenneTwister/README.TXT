Mersenne Twister based Random Number Generator

-Overview:

-Hardware requirement:
This sample requires DirectX 11 capable card, if none detected sample will use DirectX 11 Reference Emulator.

-Software requirement:
Install Visual Studio 11 from http://msdn.microsoft.com

-Running sample:
This sample generates float random numbers. 
This application is ported from CUDA SDK sample.

Mersenne Twister generate uniformly distributed random number sequences, while Box-Muller transformation is to transform into normal distribution
