Scan
This sample is also know as prefix sum. 

-Overview:
This sample is an implementation of parallel prefix sum using C++ AMP. This sample is a ported from CUDA SDK samples

-Hardware requirement:
This sample requires DirectX 11 capable card, if none detected sample will use DirectX 11 Reference Emulator.

-Software requirement:
Install Visual Studio 11 from http://msdn.microsoft.com
