Random Traveler

-Overview:
The random traveler starts at an airport, randomly takes an outgoing flight, reaches another airport, again randomly takes an outgoing flight from this airport and so on. The ample solves the following problem: given the starting airport and the number of flights the traveler takes, find the cities where the traveler is most likely to end the journey (i.e. cities with highest probabilities as final destination).

-Hardware requirement:
This sample requires DirectX 11 capable card, if none detected sample will use DirectX 11 Reference Emulator.

-Software requirement:
Install Visual Studio 12 from http://msdn.microsoft.com
