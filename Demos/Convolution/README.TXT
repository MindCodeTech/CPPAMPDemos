Convolution

-Overview:
This sample implements convolution separable filter for image progressing

-Hardware requirement:
This sample requires DirectX 11 capable card, if none detected sample will use DirectX 11 Reference Emulator.

-Software requirement:
Install Visual Studio 11 from http://msdn.microsoft.com

-Running sample:
This sample implements CPU and GPU version of Convolution. There are 2 version of GPU implementations - simple(non-tiled) and tiled. CPU version is used to verify correctness of different GPU implementations

http://en.wikipedia.org/wiki/Convolution
