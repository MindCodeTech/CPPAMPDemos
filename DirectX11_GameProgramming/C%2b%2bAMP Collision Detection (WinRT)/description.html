Introduction

This sample shows how to use C++AMP to implement collision detection between models on Windows 8.1. 

Building the Sample

The following are some essential prerequisites that should be considered before developing an application using DirectX 11.2:

  • You need to set up Microsoft Visual Studio Express 2013 for Windows 8.1 as a developer environment 
  • Make sure you set up the Windows Software Development Kit (SDK) for Windows 8.1. The Windows SDK for Windows 8.1 contains headers, libraries, and tools that can be used when creating Windows 8.1 applications.

The code in this sample can be built using Visual Studio 2013 rather than Visual Studio 2012.

  1. Start Visual Studio 2013 and select File >Open >Project/Solution.
  2. Go to the directory in which you unzipped the sample. Go to the directory named
    for the sample, and double-click the Visual Studio 2013 Solution (.sln) file.
  3. Press F6 or use Build > Build Solution to build the sample.
