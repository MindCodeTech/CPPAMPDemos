Passing pointers through C++ AMP

-Overview:
This project demonstrates an approach for passing pointers through C++ AMP runtime.
Please refer to the accompaning blog post for more information:
http://blogs.msdn.com/b/nativeconcurrency/archive/2011/12/09/passing-pointers-through-c-amp.aspx

-Hardware requirements:
This sample requires DirectX 11 capable card, if none detected sample will use DirectX 11 Reference Emulator.

-Software requirements:
Install Visual Studio 2012 from http://msdn.microsoft.com
